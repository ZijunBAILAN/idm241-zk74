Triggers

  • Hover on/off the Movie Picture.
  • Hover on/off the movie time.
  • Hover on/off the “Get Tickets” button.
  • Click the “Get Tickets” button

    • Rules

    Movie Picture:
    • Hover on: the movie picture get bigger
    • Hover on: cursor changes from default arrow to hand
    • Hover off: the movie picture get smaller
    • Hover off: cursor changes back from hand to default arrow
    Movie Time:
    • Hover on the movie time: a reminder box appears
    • Hover on: cursor changes from default arrow to hand
    • Hover off the movie time: a reminder box disappears
    • Hover off: cursor changes back from hand to default arrow
    Get Tickets:
    • Hover on the “Get Tickets” button: the button’s color change darker
    • Hover off the “Get Tickets” button: the button’s color change brighter
    • Hover on: cursor changes from default arrow to hand
    • Hover off: cursor changes back from hand to default arrow
    • Click the “Get Tickets” button: to open the “change location”.
