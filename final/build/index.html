idm-241 Zijun Kang

Get Tickets

https://www.amctheatres.com/movies

Triggers

  • Hover on/off the Movie Picture.
  • Hover on/off the heart icon.
  • Click the heart icon.
  • Hover on/off the “Get Tickets” button.
  • Click the “Get Tickets” button.
  • Hover on/off the “Search” button.
  • Hover on/off the “Exit” button.
  • click the “Exit” button.
  • Type in the “search bar.”

    • Rules

    Movie Picture:
    • Hover on: the movie picture gets darker.
    • Hover on: the heart icon and “Get tickets” button appear.
    • Hover off: the movie picture get brighter.
    • Hover off: the heart icon and “Get tickets” button disappear.
    Heart icon:
    • Hover on: cursor changes from default arrow to hand.
    • Hover off: cursor changes back from hand to default arrow.
    • First click the heart icon and the color changed to red.
    • Second click the hear icon the color changed from black to red.
    • When users click the heart icon, the database is updated.
    Get Tickets:
    • Hover on: cursor changes from default arrow to hand.
    • Hover off: cursor changes back from hand to default arrow.
    • Hover on the “Get Tickets” button: the button’s color change darker.
    • Hover off the “Get Tickets” button: the button’s color change brighter.
    • Click the “Get Tickets” button: to open the “change location” mode.
    Location:
    • Hover on the search bar, cursor changes from default arrow to caret.
    • Hover off the search bar, cursor changes back from caret arrow to default arrow.
    • Type on the search bar.
    Search icon:
    • Hover on the search icon: cursor changes from default arrow to hand.
    • Hover off: cursor changes back from hand to default arrow
    Exit button:
    • Hover on: cursor changes from default arrow to hand.
    • Hover off: cursor changes back from hand to default arrow
    • Click the exit button, the location mode closes.
