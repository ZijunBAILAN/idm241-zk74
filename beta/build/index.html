Get Tickets

https://www.amctheatres.com/movies

Triggers

  • Hover on/off the Movie Picture.
  • Click the like icon.
  • Hover on/off the “Get Tickets” button.

    • Rules

  • Hover on: the movie picture get bigger and darker.
  • Hover off: the movie picture get smaller and brighter
  • The cursor changes from a pointer to a hand pointer when hovering off.
  • Click the hear icon and the color changed to red
  • Hover on the “Get Tickets” button: the button’s color change from red to grey
  • Hover off the “Get Tickets” button: the button’s color change from grey to red.

    • Feedback
